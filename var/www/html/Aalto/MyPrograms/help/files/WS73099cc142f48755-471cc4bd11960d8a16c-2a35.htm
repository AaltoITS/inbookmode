Introduction
 
 
 

The following section describes each conversion option and their related settings. This section also contains a special section on naming conventions for FBX files that are converted into other FBX versions. We recommend leaving the default conversion options as they are, except in special cases. For example, if your scene is large and you do not need textures or materials imported, you would alter the default conversion settings.

NoteThese descriptions apply to both the Macintosh and Windows versions of the FBX Converter.
