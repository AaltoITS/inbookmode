Creating a conversion preset
 
 
 

You can create a preset that you can apply when you wish to use the same conversion options repeatedly.

To create a conversion preset:

  1. From the main FBX Converter menu, select Presets.
  2. Select Edit.
  3. A list of all applicable options for import and export appears.

    Select any desired options to uses these as your preset settings.

  4. Select Preset > Save. An Fbx Converter Preset Name window appears for you to enter the name of your preset.

    FBX Converter Preset name window

    To edit a preset, select Edit from the Presets menu, make your changes and click Save.
